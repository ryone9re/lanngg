# lanngg

ChatGPTに設計してもらった言語を自分で実装していくことで言語実装の経験を積んでいく

```txt
// 変数宣言
var x: int;
var y: float;
var flag: bool;

// 配列宣言
var arr: int[10];

// 構造体定義
struct Point {
    var x: int;
    var y: int;
}

// 構造体変数宣言
var p: Point;

// 関数定義
fn add(a: int, b: int) int {
    return a + b;
}

// main関数
fn main() int {
    var a: int = 10;
    var b: int = 20;
    var result: int;

    result = add(a, b);
    return 0;
}
```
